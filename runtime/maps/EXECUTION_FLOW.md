# Execution Flow

This map records route boundaries only. It does not implement or expand routes.

1. /session establishes session evidence.
2. /continuity binds continuity evidence.
3. /authority creates authority only through the existing runtime route.
4. /compile derives the exact AEO.
5. /validate returns VALID or NULL.
6. /execute remains validation-bound.
7. /proof persists proof through the existing proof boundary.

No new POST route is declared by this file.
